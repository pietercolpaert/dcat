A file host publishing LDES feeds for DCAT-AP exports from data portals across the World

These feeds implement the DCAT-AP Feeds specification built on top of the SEMIC LDES specification. You can use an LDES client or an LDES to Service integration to fully replicate the dataset to your own system.
